## 🌟 Overview

**Sway** is a premium event catering and hospitality platform serving Palestine and Jordan. We specialize in creating unforgettable experiences for weddings, diplomatic events, corporate gatherings, and private celebrations.

### 🎯 Mission

> *"From Sea to River, we craft extraordinary moments with impeccable service, exquisite beverages, and culinary excellence."*

Since 2021, Sway has been setting new standards in the event hospitality industry, serving everyone from ambassadors and ministers to intimate family celebrations.

## 🚀 Quick Start

### **Installation**

```bash
# Clone the repository
git clone https://github.com/yourusername/sway-luxury.git

# Navigate to project directory
cd sway-luxury

# Open in browser
open index.html
```

### **Deployment**

#### **Netlify** (Recommended)
```bash
# Install Netlify CLI
npm install -g netlify-cli

# Deploy
netlify deploy --prod
```

#### **GitHub Pages**
```bash
# Push to GitHub
git add .
git commit -m "Initial commit"
git push origin main

# Enable GitHub Pages in repository settings
```

#### **Vercel**
```bash
# Install Vercel CLI
npm i -g vercel

# Deploy
vercel --prod
```

## 🔧 Customization Guide

### **Change Colors**
```css
/* Update CSS variables */
--primary-gold: #DAA520;
--light-gold: #FFD700;
--dark-bg: #0a0a0a;
```

### **Update Content**
```html
<!-- Edit text directly in HTML -->
<h1>Your New Title</h1>
<p>Your new description</p>
```

### **Add Services**
```html
<div class="service-card">
    <h3>New Service</h3>
    <p>Service description here</p>
</div>
```

### **Modify Animations**
```css
/* Adjust animation duration */
transition: all 0.4s ease;

/* Change hover effects */
.service-card:hover {
    transform: translateY(-10px);
}
```
